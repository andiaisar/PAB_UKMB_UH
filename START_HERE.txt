┌─────────────────────────────────────────────────────────────────┐
│ STEP 1: BACA DOKUMENTASI                                        │
└─────────────────────────────────────────────────────────────────┘

Mulai dari file ini (urutan yang disarankan):

1. README.md          - Baca overview & fitur utama
2. CHECKLIST.md       - Ikuti checklist setup
3. QUICKSTART.md      - Panduan cepat jika terburu-buru
4. README_PANDUAN.md  - Baca untuk detail lengkap

┌─────────────────────────────────────────────────────────────────┐
│ STEP 2: SETUP FIREBASE CONFIG                                   │
└─────────────────────────────────────────────────────────────────┘

File: src/firebase.js

1. Buka https://console.firebase.google.com
2. Pilih project Anda
3. Klik ⚙️ Settings → Project settings
4. Scroll ke "Your apps" → Web app
5. Copy config ke src/firebase.js

┌─────────────────────────────────────────────────────────────────┐
│ STEP 3: SETUP GOOGLE SHEET                                      │
└─────────────────────────────────────────────────────────────────┘

1. Buat Google Sheet dengan header:
   nama | nim | fakultas | prodi | hp

2. Isi dengan data peserta (atau gunakan sample-data.csv)

3. Publish sebagai CSV:
   File → Share → Publish to web → Format: CSV

4. Copy URL ke src/components/SyncButton.jsx (baris 14)

┌─────────────────────────────────────────────────────────────────┐
│ STEP 4: SETUP FIRESTORE RULES                                   │
└─────────────────────────────────────────────────────────────────┘

1. Buka Firebase Console → Firestore Database
2. Klik tab "Rules"
3. Copy paste dari file: firestore.rules
4. Klik "Publish"

┌─────────────────────────────────────────────────────────────────┐
│ STEP 5: RUN THE APP                                             │
└─────────────────────────────────────────────────────────────────┘

Terminal:
```bash
npm run dev
```

Browser:
```
http://localhost:5173
```

┌─────────────────────────────────────────────────────────────────┐
│ STEP 6: FIRST SYNC                                              │
└─────────────────────────────────────────────────────────────────┘

1. Buka aplikasi di browser
2. Klik tombol "🔄 Sync dari Google Sheet"
3. Tunggu pesan sukses
4. Data peserta akan muncul!

🔥 FITUR KUNCI: MERGE TRUE

Saat sync dari Google Sheet, kode menggunakan:

```javascript
await setDoc(docRef, dataToSave, { merge: true });
```

KEUNTUNGAN:
✅ Peserta BARU akan ditambahkan ke Firestore
✅ Data profil peserta LAMA akan diupdate (nama, fakultas, dll)
✅ Status checkbox (tahap_1 - tahap_5) peserta lama TIDAK akan direset
✅ Aman untuk sync berulang kali tanpa kehilangan data checkbox

CONTOH SKENARIO:
- Anda sudah sync 100 peserta
- Sudah dicentang status tahap beberapa peserta
- Kemudian ada 10 peserta baru di Google Sheet
- Anda sync lagi
- HASIL: 10 peserta baru ditambahkan, status checkbox peserta lama TETAP ✅

📊 STRUKTUR DATA FIRESTORE:

Collection: peserta
Document ID: NIM (sebagai identifier unik)

Fields:
- nama      : string
- nim       : string
- fakultas  : string
- prodi     : string
- hp        : string
- tahap_1   : boolean (default: false)
- tahap_2   : boolean (default: false)
- tahap_3   : boolean (default: false)
- tahap_4   : boolean (default: false)
- tahap_5   : boolean (default: false)

🐛 TROUBLESHOOTING CEPAT:

Problem: Firebase is not defined
✅ Solution: Isi config di src/firebase.js

Problem: Sync gagal / CORS error
✅ Solution: Pastikan Google Sheet sudah dipublish sebagai CSV

Problem: Checkbox tidak bisa diklik
✅ Solution: Cek Firestore rules allow write: true

Problem: Data tidak muncul di tabel
✅ Solution: Klik "Sync dari Google Sheet" terlebih dahulu

Problem: Recharts error
✅ Solution: Sudah terinstall, cek dengan: npm list recharts
